　　原标题：孟山都前华裔雇员因间谍罪在美被捕，耿爽：支持美方依法秉公处理，但反对借题发挥

\ \

　　[环球时报-环球网报道\ 记者\ 李司坤]据美国之音报道，孟山都公司一名前华裔雇员向海涛11月21日被联邦大陪审团以一项共谋经济间谍罪、三项经济间谍罪、一项共谋盗窃商业秘密罪和三项窃取商业秘密罪起诉。在22日，有路透社记者就此事提问，为此，发言人耿爽在回应中表示，美方若依法秉公地进行处理，中方没有意见，但坚决反对美方借题发挥。

\

　　耿爽在回应中称，你说的具体情况我不了解，但我想，无论任何人，中国公民还是美国公民，如果在美国有违法的行为，美方依法秉公地进行处理，我们对此没有意见，但我们坚决反对美方借题发挥，把一个普通的孤立的案件说成是中方有组织、有计划从美方窃取知识产权。

\

　　耿爽强调，第一，中国的科技成就一不靠偷，二不靠抢，是中国人民凭着智慧和汗水拼搏奋斗而来的；第二，中美加强科技交流合作符合双方共同利益，也有利于推动科技进步，造福全人类。
